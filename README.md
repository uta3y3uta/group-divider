# グループ分けアプリ

人数と班数を入れるだけで均等に自動分けできるシンプルなグループ分けアプリです。
席替えの班版として、またペア決めにも使えます。

## 機能

- **初期設定**: メンバー登録（名前・性別・タグ任意）、まとめて追加にも対応
- **ランダム**: 班数または1班の人数で指定、「男女混合」「前回と違うメンバー」条件指定
- **手動**: 名前カードをドラッグ&ドロップで班に配置
- グループ名は任意で編集可
- データはブラウザの LocalStorage に保存

## 使い方

1. 「初期設定」タブでメンバーを登録
2. 「ランダム」タブで自動振り分け、または「手動」タブでドラッグして配置
3. 各班の名前は空欄でもOK（自動で「グループ1」等になる）

## ローカルで開く

```
index.html をブラウザで開くだけ
```

ビルド不要、依存ライブラリなし。
